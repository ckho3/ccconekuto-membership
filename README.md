# NTTドコモ ウェブサイトクローン

このプロジェクトは、NTTドコモの公式ウェブサイト（https://www.docomo.ne.jp/）のレイアウトとデザインを再現したものです。

## 機能

### 実装済みの機能

1. **レスポンシブヘッダー**
   - ドコモロゴ
   - ナビゲーションメニュー
   - My docomo / オンラインショップボタン
   - モバイルメニュートグル

2. **カルーセル（スライダー）**
   - 自動再生機能（5秒間隔）
   - 手動スライド切り替え（前へ/次へボタン）
   - インジケーター（ドット）でのナビゲーション
   - 一時停止/再生ボタン
   - スムーズなフェードアニメーション

3. **よく利用されるサービスセクション**
   - グリッドレイアウト（8アイテム）
   - ホバーエフェクト（浮き上がり）
   - アイコン付きサービスリンク

4. **目的から探すセクション**
   - 6つのカテゴリカード
   - グラデーション背景
   - ホバーアニメーション

5. **タブ切り替えセクション**
   - 4つのタブ（モバイル、インターネット、スマートライフ、電気・ガス）
   - スムーズなタブ切り替え
   - 各タブに対応するコンテンツカード

6. **フッター**
   - SNSリンク（X, LINE, Instagram, YouTube）
   - サイトマップリンク
   - コピーライト表示

### アニメーション機能

- **カルーセルアニメーション**: スライドのフェードイン/アウト
- **スクロールアニメーション**: 要素の表示時にフェードイン
- **ホバーエフェクト**: カードの浮き上がりとグロー効果
- **パララックス効果**: スクロール時の背景移動
- **スムーススクロール**: アンカーリンクのスムーズな移動

### デザイン特徴

- **カラーパレット**:
  - メインカラー: #e60012（ドコモレッド）
  - 背景: #fff（白）、#f8f8f8（ライトグレー）
  - テキスト: #333（ダークグレー）

- **レスポンシブデザイン**:
  - デスクトップ、タブレット、モバイル対応
  - ブレークポイント: 767px

## セットアップ

### 必要な環境

- Node.js (v14以上)
- npm

### インストール

```bash
cd docomo-clone
npm install
```

### 起動方法

```bash
npm start
```

または

```bash
node server.js
```

サーバーが起動したら、ブラウザで以下のURLにアクセスしてください：

```
http://localhost:3000
```

### 開発モード

```bash
npm run dev
```

## カスタマイズ

### 色の変更

`public/styles.css` ファイルで以下の部分を編集してください：

```css
/* メインカラー（ドコモレッド） */
background-color: #e60012;
color: #e60012;
border-color: #e60012;
```

### カルーセルの速度変更

`public/script.js` のCarouselクラス内で変更：

```javascript
this.interval = setInterval(() => {
    this.nextSlide();
}, 5000); // 5000ms = 5秒（この数値を変更）
```
